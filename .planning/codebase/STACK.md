# Tech Stack

## Frontend
- **Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS & Vanilla CSS
- **Components**: Shadcn UI (Radix UI primitives)
- **State/Caching**: TanStack React Query (@tanstack/react-query)
- **Routing**: React Router DOM (v6)

## Backend
- **Framework**: FastAPI (v0.103.1)
- **Language**: Python
- **Database**: SQLite (via built-in sqlite3)
- **Image Processing**: Pillow (v10.0.1)
- **Hashing**: ImageHash (v4.3.1)
- **Server**: Uvicorn

## Tooling
- **Linting**: ESLint, Lovable Tagger
- **Testing**: Vitest, Playwright
